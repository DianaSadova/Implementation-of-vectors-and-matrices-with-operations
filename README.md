# Реализация векторов и матриц с операциями в C++

## Тема:
**Реализация и работа с классами векторов и матриц в C++ с поддержкой арифметических операций, включая вычисление определителя и обратной матрицы.**

---

## Описание кода

### Основные компоненты:

1. **Класс `vect`** - реализует векторы в многомерном пространстве.
2. **Класс `matr`** - реализует квадратные матрицы.
3. Реализованы арифметические операции для векторов и матриц:
   - Сложение (+)
   - Вычитание (-)
   - Умножение (*).
4. Реализованы дополнительные функции:
   - Вычисление определителя матрицы.
   - Вычисление обратной матрицы.

### Описание классов и функций

#### Класс `vect`:
- **Поля:**
  - `dim` - размерность вектора.
  - `v` - динамический массив для хранения компонентов.
- **Конструкторы:**
  - Создание нулевого вектора заданной размерности.
  - Копирование вектора.
- **Операторы:**
  - Присваивание `=`.
  - Арифметические операции: сложение, вычитание, унарный минус.
  - Скалярное произведение и умножение на число.

#### Класс `matr`:
- **Поля:**
  - `dim` - размерность матрицы.
  - `a` - динамический двумерный массив для хранения элементов матрицы.
- **Конструкторы:**
  - Создание нулевой матрицы заданной размерности.
  - Копирование матрицы.
- **Операторы:**
  - Присваивание `=`.
  - Арифметические операции: сложение, вычитание, унарный минус, умножение на число, умножение матриц.
  - Умножение матрицы на вектор.

#### Вычисление определителя (`determinant`):
- Рекурсивный алгоритм, основанный на разложении по строке (метод миноров).
- Использует подматрицы для вычисления детерминанта.

#### Вычисление обратной матрицы (`inverse`):
- Использует определитель для проверки вырожденности.
- На основе метода алгебраических дополнений.

---

## Пример использования

```cpp
int main() {
    vect v1(3); // Создаем вектор размерности 3
    vect v2(3);

    v1.v[0] = 1; v1.v[1] = 2; v1.v[2] = 3;
    v2.v[0] = 4; v2.v[1] = 5; v2.v[2] = 6;

    vect v3 = v1 + v2; // Сложение векторов
    cout << v3 << endl;

    matr m1(2);
    matr m2(2);

    m1.a[0][0] = 1; m1.a[0][1] = 2;
    m1.a[1][0] = 3; m1.a[1][1] = 4;

    m2.a[0][0] = 5; m2.a[0][1] = 6;
    m2.a[1][0] = 7; m2.a[1][1] = 8;

    matr m3 = m1 * m2; // Умножение матриц
    cout << m3 << endl;

    double det = determinant(m1, 2); // Определитель матрицы
    cout << "Determinant: " << det << endl;

    return 0;
}
```
